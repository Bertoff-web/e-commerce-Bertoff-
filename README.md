# e-commerce-Bertoff-
O Bertof é um sistema de gestão para lojas, clientes e produtos de forma integrada. O público-alvo são pequenos comerciantes e empreendedores que vendem diretamente para pessoas físicas. Seu principal propósito é organizar o fluxo de vendas e facilitar o gerenciamento diário da loja, garantindo eficiência e precisão nas operações.
